# Attribute Completion GNN for Collaborative Filtering

This repository contains a PyTorch implementation of our 
Attribute Completion–based Graph Neural Network Collaborative Filtering model.

## Datasets Supported
- Douban Book
- FilmTrust
- Last.fm
- Yelp

## Usage
```bash
python ac_gnn_cf.py --dataset yelp --data_dir /path/to/yelp
